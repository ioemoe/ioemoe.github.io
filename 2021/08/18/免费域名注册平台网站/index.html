提供免费域名的网站

在初期建站时候，我们通常会选择寻找免费的域名，虚拟主机等等，这里我提供一些免费注册域名的网站。供大家查考也是为了方便自己查阅。

1、免费域名有哪些形式？

（1）顶级域名

目前，提供真正免费顶级域名的互联网公司还比较少，可以说是凤毛麟角。其中，最知名的就是由Freenom运营的.tk，.ml，.cf，.gq，.ga五大顶级免费域名，而且用户量非常大，还有一些其他厂商提供的 .one 和 .me 等免费顶级域名。

（2）二级域名

相对于免费顶级域名，大部分提供的免费域名都是二级域名，比如 pp.ua，.eu.org，nom.za 等，而且也是被广泛应用的，多是一些边缘产业在用，真正用来建站的很少。

（3）免费赠送的域名

除了上述专业提供免费域名的平台之外，还有很多VPS服务商提供增值服务包。你在购买他们VPS服务时，会赠送你免费域名的附加服务，例如 Hostinger、Bluehost、ipage.com、ionos.com 等。鉴于增值服务是需要先购买服务，然后才有赠送，其实跟付费差不多了。

当然，免费域名用来建站其实是不划算的，想要长期运营一家网站还是建议用付费域名比较安全稳定。如果你是打算用来建站的话，我推荐您购买付费域名。

2、免费顶级域名

（1）Freenom

Freenom 是全球最大、最知名的免费域名注册网站，也是世界上第一个且唯一的专业免费域名提供商，网站支持多国语言（包括简体中文），所以中文用户也可以无语言障碍地畅快使用。Freenom 采用最新的任播（ AnyCast ）云技术，可保证其管理下所有域名的稳定性和性能。 Freenom 拥有战略上值得信赖的安全合作伙伴，如推特（ Twitter ）、Internet Identity 、卡巴斯基（Kaspersky ）和其他 40 多家合作伙伴，能够提供最先进的反滥用技术，以保持免费域名的安全性，并确保所有互联网用户的安全。2013年11月，Architelos旗下”.TK”拥有超过2000万个域名，为最大顶级域名，是世界上最安全的域名扩展名。

域名类型：顶级域名top-level domain (TLD)
注册地址：https://www.freenom.com 或 http://www.dot.tk
所属国家：Equatorial Guinea (赤道几内亚)
DNS：提供自定义DNS域名服务器
备注：.tk，.ml，.cf，.gq，.ga 域名都由Freenom运营

（2）.me

域名类型：顶级域名top-level domain (TLD)
注册地址：https://nc.me
DNS：提供自定义DNS域名服务器
备注：由 Namecheap 运营，.me 域名注册对学生用户限时免费。

（3）.co.uk/.uk

域名类型：顶级域名top-level domain (TLD)
注册地址：https://www.names.co.uk
所属国家：United Kingdom (英国)
DNS：提供自定义DNS域名服务器
备注：免费注册 .co.uk、.uk 域名，仅首年免费

3、免费二级域名

（1）.pp.ua

域名类型：二级域名second-level domain (SLD )
注册地址：https://nic.ua/en/domains
所属国家：Ukraine (乌克兰)
DNS：提供自定义DNS域名服务器
优点：长期免费

注册 .pp.ua 域名的简明步骤：

  1. 在 https://nic.ua/en/domains 搜索以 pp.ua 后缀的域名，加入到购物车，然后 checkout 去结账；
  2. 注册，验证邮箱；
  3. 激活域名；
    a、登录 https://apu.drs.ua/en/，使用手机号激活账户；
    b、Telegram激活：搜索 @ppuabot

（2）.eu.org

域名类型：二级域名second-level domain (SLD )
注册地址：https://nic.eu.org
所属国家：United Kingdom (英国)
DNS：提供自定义DNS域名服务器
优点：长期免费
备注：由 United Kingdom 的黑客组织运营，提供免费域名注册，虽然界面简陋，但能注册且真正免费，人工审核时间相对长。

（3）.ze.cx

域名类型：二级域名second-level domain (SLD )
注册地址：https://www.azote.org
所属国家：France(法国)
优点：长期免费
备注：免费提供.asso.st，.fr.nf，.fr.cr，.ze.cx，.infos.st 二级域名注册。

（4）.nom.za

域名类型：二级域名second-level domain (SLD )
注册地址：https://www.nom.za
所属国家：South Africa (南非)
DNS：提供自定义DNS域名服务器
优点：长期使用
备注：人工审核，时间较长。

（5）.org.uk

域名类型：二级域名second-level domain (SLD )
注册地址：https://www.fasthosts.co.uk
所属国家：United Kingdom (英国)
DNS：提供自定义DNS域名服务器
备注：免费注册 .co.uk、me.uk、org.uk、.uk 域名，仅首年免费

（6）.rr.nu

域名类型：二级域名second-level domain (SLD )
注册地址：https://www.sitelutions.com
所属国家：Niue(纽埃)
备注：免费注册，但DNS解析、Redirect需要付费。
提供 rr.nu，kwikto.com，myfw.us,lookin.at、PassingGas.net、MyRedirect.us、OnTheWeb.nu、IsTheBe.st、ByInter.net、FindHere.org、OnTheNetAs.com、ByInter.net、FindHere.org、OnTheNetAs.com、UglyAs.com、AsSexyAs.com、PassAs.us、PassingGas.net、AtHisSite.com、AtHerSite.com、IsGre.at、Lookin.At、BestDeals.At、LowestPrices.At 等几十个二级域名注册。

注册 .rr.nu 域名简明步骤：

  1. 注册账号，登录
  2. 菜单 -> ACCOUNT CONTROL -> URL and DNS
    或页面：DNS & Redirection -> URL & DNS Control Center

（7）.ipq.co

域名类型：二级域名second-level domain (SLD )

所属国家：Republic of Colombia (哥伦比亚共和国)
